# [Group-6] 中部市場蔬菜作物價格預測
本研究探討氣候因素對蔬果價格的影響，透過分析氣溫、降雨量、日照時數等資料，建立價格預測模型。希望找出影響價格波動的關鍵天氣因素，協助消費者、農民與產業做出更有效的決策。

    | 中價  |      數值型       | 元 / 公斤     |
